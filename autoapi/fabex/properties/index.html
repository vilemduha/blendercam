update_operation(self, context)[source]#

Update the CAM operation based on the current context.

This function retrieves the active CAM operation from the Blender context and updates it using the updateRest function. It accesses the active operation from the scene’s CAM operations and passes the current context to the updating function.

Parameters:

context – The context in which the operation is being updated.
